# NativeUI

**React Native 组件库** - 基于 @shopify/restyle，参考 Chakra UI 设计理念。

[快速开始](/guide/getting-started) · [主题配置](/guide/theme) · [迁移指南](/guide/migration)

## 特性

- **类型安全**：基于 TypeScript，完整的类型推导和自动补全
- **主题驱动**：通过 @shopify/restyle 实现主题化，支持亮色/暗色模式切换
- **开箱即用**：30+ 常用组件，覆盖布局、排版、表单、数据展示、反馈等场景
