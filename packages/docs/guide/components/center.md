# Center 组件

居中布局容器，子元素在水平和垂直方向上居中显示。

## 引入

```tsx
import { Center } from 'kra-ui';
```

## 基本用法

```tsx
<Center width={200} height={200} backgroundColor="primaryLight" borderRadius="m">
  <Text>居中内容</Text>
</Center>
```

## Props

| 属性        | 类型 | 说明              |
| ----------- | ---- | ----------------- |
| ...BoxProps | -    | 继承所有 Box 属性 |
